# Frontend API 연동 문서

## 1. 로그인(Login)

| API              | 설명                   | 연동 방식                           | 비고                     |
| ---------------- | ---------------------- | ----------------------------------- | ------------------------ |
| POST /auth/login | 이메일/비밀번호 로그인 | 로그인 버튼 클릭 시 Axios POST 호출 | 실패 시 오류 메시지 표시 |

## 2. 별자리(Zodiac)

| GET /zodiac | 별자리 리스트 조회 | 페이지 진입 시 Axios GET 호출 | 로딩 시 Skeleton UI 표시 가능 |
