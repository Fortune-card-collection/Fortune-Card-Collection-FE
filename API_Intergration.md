# Frontend API 연동 문서


## 1. 로그인(Login)
| API              | 설명                   | 연동 방식                           | 비고                     |
| ---------------- | ---------------------- | ----------------------------------- | ------------------------ |
| GET /auth/kakao | 로그인 후 카카오에서 받은 `code` 값을 백엔드로 전달하여 cookie 값을 받음 | 로그인 완료 시 사용자 인증 처리 | 향후 쿠키로 로그인 상태 확인 |


## 2. 별자리(Zodiac)
| API         | 설명               | 연동 방식                     | 비고                          |
| ----------- | ------------------ | ----------------------------- | ----------------------------- |
| GET /horosocopes/{별자리}/{시간대} | 선택된 별자리 운세를 카드로 보여줌 | 선택된 별자리, 선택된 시간대의 운세값 출력 | 페이지에서 자유롭게 시간대 별자리 선택 가능 |


## 3. 생년월일(Birth)
| API         | 설명               | 연동 방식                     | 비고                          |
| ----------- | ------------------ | ----------------------------- | ----------------------------- |
| GET /users/me | 로그인한 사용자의 정보에서 값이 저장되어 있으면 자동 입력 | 사용자의 정보 반환 | |
| PATCH /users/me | 사용자가 검색하고 싶은 생년월일 정보를 사용자 정보에 업데이트 | 사용자 정보 업데이트 | |
| POST /fortune/personal/today | 사용자 정보를 기반으로 AI API를 이용해 운세 값 받음 | Chat API 연동 | 사용자 1인당 1일 1회 가능 |
| GET /fortune/personal/today | 이미 사용자가 POST를 통해 오늘의 운세를 받앗을 경우 | 사용자가 받았던 오늘의 운세값 출력 | POST에서 1일 2번 이상 실행 시 수행 |


## 4. 명언(Quote)
| API         | 설명               | 연동 방식                     | 비고                          |
| ----------- | ------------------ | ----------------------------- | ----------------------------- |
| GET /quotes/random | quotes에서 랜덤숫자 3개를 받음 | 난수 3개 반환 | |
| POST /quotes/select | 선택한 카드를 서버에게 전송 | 선택된 카드의 명언ID 전달 | |
| GET /quotes/{명언ID} | 명언 상세보기 | 선택된 명언값 출력 |  |
